Kanri has a **Duplicate** feature that creates a copy of a member's profile. This is useful when you need to register a family member or someone in similar circumstances who shares most of the same details (address, contacts, family status, and so on).

Kanri does not currently have an automated merge tool. If two separate profiles exist for the same person, you'll need to handle that manually. This page covers both situations.

## Duplicating a member profile

Use **Duplicate** when you want to create a new member whose profile is largely the same as an existing one — for example, registering a second child in the same family.

1. Open the existing member's profile.
2. Click the **Duplicate** icon (two overlapping pages) in the top right of the **Information** card.
3. Review the pre-filled form. The following fields are copied from the original:
   - Last name
   - Family status
   - Sex
   - Birthday
   - Can manage self
4. **First Name is intentionally left blank** — you must enter it before saving.
5. Phone and email are also left blank (since these must be unique per member).
6. Click **Duplicate** to create the new profile.

What gets copied to the new profile:

- All of the original's **contacts** (including family relationships)
- All of the original's **notes**
- The original's address

What is **not** copied:

- [Memberships](/members/memberships/)
- [Subscriptions and billing information](/members/subscriptions/)
- [Attendance history](/members/attendance/)
- [Gallery files](/members/gallery/)
- [Interactions](/members/interactions/)

:::note
After duplicating, you'll need to set up memberships and billing for the new member separately, just as you would for any new member.
:::

## Handling duplicate members (same person, two profiles)

If the same person was accidentally added twice, you won't find a one-click merge button in Kanri. Here's how to handle it manually:

### 1. Identify which profile to keep

Open both profiles and compare them. Choose the one with more complete information as the profile to keep. Key things to check:

- Which has a [membership](/members/memberships/)?
- Which has a [subscription](/members/subscriptions/)?
- Which has attendance history?
- Which has the correct email and phone number?

### 2. Move any important notes

Go to the duplicate profile and copy the content of any [notes](/members/notes/) that aren't on the profile you're keeping. Add them as new notes on the correct profile.

Do the same for any [interactions](/members/interactions/) — copy the key details into a new interaction on the correct profile, noting that the entry was transferred from a duplicate.

### 3. Update the profile you're keeping

Make sure the profile you're keeping has the correct email, phone number, address, and photo. Use [Edit a Member](/members/edit-member/) to update anything that's missing.

### 4. Delete the duplicate

Once you're confident the correct profile is complete, you can delete the duplicate from the member's profile page. Deletion is permanent.

## Preventing duplicates

The most common cause of duplicates is adding a member manually while they already submitted a lead form or booked a trial. To reduce the chances of this happening:

- Search by email before adding a new member manually.
- Use the [auto-link accounts](/settings/behaviors/general-behaviors/) behavior, which automatically links portal accounts to member profiles when emails match.
- When a lead comes in through your [lead page](/lead-management/lead-page/), check the [Leads list](/lead-management/view-leads/) before creating a new member manually.
