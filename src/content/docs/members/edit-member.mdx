---
title: Edit a Member
description: How to update a member's details, photo, and address.
---

You can update a member's personal details at any time. The edit page lets you change the fields you filled in when the profile was created. Address and photo are updated on separate pages.

## Before you start

You need the **Manage Members** permission to edit member profiles.

## Edit basic details

1. Open the member's profile by clicking their name in the [Members list](/members/).
2. Click the **Edit** icon (pencil) in the top right of the **Information** card — or go to the **Members** list and click the **Edit** icon directly in the row.
3. Update the fields you want to change.
4. Click **Save**.

The following fields can be updated on the edit page:

- First name, last name
- Family status
- Sex
- Birthday
- Phone number
- Email address
- Can manage self

:::note
The member's **ID** and **member number** cannot be changed. The member number is assigned automatically when the profile is created. See [Member Number](/members/member-number/) for more context.
:::

## Update the address

The member's address is edited on a separate page.

1. Open the member's profile.
2. Click the **Edit** icon in the **Address** card.
3. Fill in the address fields: Line 1, Line 2, City, Province/State, Postal Code, and Country.
4. Click **Save Address**.

The Country field uses a searchable list — type the country name to find it.

## Upload or change a photo

A profile photo makes it easier to identify members on the kiosk and in the app. Photos are shown as a circle cropped from the centre of the image, so a clear face shot works best.

1. Open the member's profile.
2. Click the **camera/edit icon** next to the current photo in the Information card.
3. Upload a new photo. You can drag and drop an image file, or choose one from the member's [gallery](/members/gallery/) if files have already been uploaded there.
4. Click **Save**.

:::tip
For the best kiosk experience, use a portrait photo with the member's face centred. Kanri automatically generates a thumbnail for faster loading at the kiosk.
:::

To remove an existing photo, navigate to the photo page and use the remove option. Selecting a new photo automatically replaces the old one.

## What can't be edited

- **Member ID** — a system-generated identifier used internally.
- **Member number** — assigned automatically at creation. See [Member Number](/members/member-number/).
- **Membership history** — past memberships are permanent records. Use the [Memberships](/members/memberships/) section to make changes.
- **Subscription history** — past payments and subscription records cannot be edited. Use the [Subscriptions](/members/subscriptions/) section for current changes.
