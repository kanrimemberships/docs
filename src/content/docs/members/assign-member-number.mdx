description: How to set a member number so a member can sign in at the kiosk using a number instead of searching their name.

## What this page will cover

- What a member number is (a short numeric ID for kiosk sign-in)
- How members use their number at the kiosk (type number → select class → signed in)
- Where to set a member number on the member's profile
- Number format requirements (numeric only, must be unique within the school)
- What to do if a member forgets their number
