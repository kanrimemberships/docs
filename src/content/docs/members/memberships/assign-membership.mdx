Assign a membership when a member joins a program.

## Before you start

Make sure:

1. The [program](/programs/create-program/) already exists.
2. Your school has the membership status you want to use.
3. The member does not already have a membership in that same program.

## Assign a membership

1. Open the member from the [Members list](/members/).
2. Click **Memberships**.
3. Click **New Membership**.
4. Select the **Program**.
5. Select the current **Level** if your school uses [Levels](/programs/levels/).
6. Select the **Status**.
7. Enter **Sessions Remaining** if you track classes in blocks.
8. Enter **Last Promotion Date** if needed.
9. Click **Create**.

## Know what each field means

| Field | What it means |
| --- | --- |
| Program | Required. The program the member is enrolled in. |
| Level | Optional. Only used if your school has levels turned on. |
| Status | Required. A school-defined membership status. |
| Sessions Remaining | Optional. Leave blank if you do not track remaining sessions. |
| Last Promotion Date | Optional. Useful when moving existing students into Kanri. |

:::caution
A member can only have one membership in each program. For example, they cannot have two separate Adult Karate memberships at the same time.
:::

:::tip
If you are importing existing students, set the current level and last promotion date right away. This makes your [Promotions report](/programs/levels/promotions-report/) more accurate.
:::
