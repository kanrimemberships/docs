description: How to enrol a member in a program by creating a membership.
---

## What this page will cover

- Prerequisites: the program must exist before you can assign a membership
- Going to the Memberships tab on a member's profile
- Selecting the program
- Setting the start date
- Selecting an initial level (if the program uses levels)
- Setting the membership status (usually Active)
- What to do next: set up a subscription for billing
