---
title: Change Membership Status
description: How to update a member's membership status (e.g. from Active to On Hold).
---

## What this page will cover

- When to change a status vs when to end a membership entirely
- How to change a membership status (click status badge → select new status)
- Common status changes and when to use them:
  - Active → On Hold (member taking a break, still enrolled)
  - Active → Inactive (member has left)
  - Inactive → Active (member returning)
- Whether status changes affect billing (they don't automatically — subscriptions are separate)
- How behaviors can automatically change statuses when a subscription is cancelled
