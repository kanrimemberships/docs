## What this page will cover

- **Single vs Multiple subscription mode**: whether a member can have one active subscription at a time or multiple simultaneously. Default is Single (most schools only need one).
- **Billing plan mode (Single vs Multiple prices)**: whether each billing plan has one price or can have multiple prices (e.g. Adult $120/mo and Junior $80/mo on the same plan). Most schools can use Single.
- **Program access gating**: when enabled, a member's active subscriptions determine which programs they can sign in to at the kiosk. Only enable this if you need to restrict access by subscription.
- **Subscription receipts**: whether members automatically receive an email receipt after each successful payment.
