## What this page will cover

- Booking window days: how many days in advance members can book into a class
- Trial booking window days: how many days in advance a prospective member can book a trial through the Lead Page
- Same-day booking: whether members can book a class on the day it runs
- How these settings interact with the Lead Page and the Member Portal
