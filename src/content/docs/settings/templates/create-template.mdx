## What this page will cover

- Going to Settings → Templates → New Template
- Setting the template name (internal use only, members never see this)
- Choosing the type: Email or SMS
- For Email: setting the subject line and body
- For SMS: body only (no subject)
- Using placeholders in the body (e.g. "Hi `&#123;&#123;first_name&#125;&#125;`,")
- Saving the template
- Editing and deleting templates
- Tip: create templates for your most common messages — payment reminders, class cancellations, promotions
