## What this page will cover

- What a vanity URL is (a custom subdomain, e.g. springfieldkarate.kanri.app)
- How vanity URLs are used (Lead Page, Member Portal, kiosk URL all use the vanity subdomain)
- Setting your vanity URL in Settings → Branding
- Rules: must be unique, lowercase, no spaces, letters and numbers only
- What happens after you set it (old URLs still redirect)
- Changing your vanity URL later (and how it affects shared links)
