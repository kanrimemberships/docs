title: Create a Custom Role
description: How to create a new role with specific permissions in Kanri.
---

## What this page will cover

- Going to Settings → Roles → New Role
- Naming the role (e.g. "Instructor", "Front Desk", "Administrator")
- Selecting permissions: going through each permission group and enabling what this role should have
- Saving the role
- Editing permissions after the role is created
- Tip: start restrictive — it's easier to add permissions than to deal with someone having too much access
