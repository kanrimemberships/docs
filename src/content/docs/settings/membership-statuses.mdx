## What this page will cover

- What membership statuses are (the labels you assign to memberships, e.g. "Active", "Suspended", "On Hold", "Trial")
- The default "Active" status and why it matters for reports (only "active" statuses count toward active member counts)
- Creating a new status: setting the name, and whether it counts as "active"
- Editing a status name
- Reordering statuses
- Deleting a status (only possible if no memberships are using it)
- Setting the default status for new memberships
