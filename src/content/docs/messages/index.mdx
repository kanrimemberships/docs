## What this page will cover

- Message types: Email vs SMS — when to use each
- Individual messages (from a member's profile) vs broadcast messages (to many members)
- How message history is stored (per member and school-wide)
- Deliverability basics: what affects whether emails land in the inbox vs spam
- SMS: what it requires (a phone number set up in Settings → Branding → Phone Number)
- Message templates: saving time on common messages
- Groups: targeting a broadcast at a specific collection of members
