## What this page will cover

- What a group is (a named collection of members — currently used for targeting broadcasts, but designed as a general-purpose grouping tool that will be used for more features in the future)
- Two types of assignments that make up a group:
  1. **Member assignments** — specific members added directly to the group
  2. **Program assignments** — all active members of a program, optionally filtered by a level range (min and max level)
- Groups are dynamic: program assignments reflect current membership, not a snapshot
- How groups are used: as a broadcast recipient target in Messages → Compose
- Viewing who is in a group (current members shown based on assignments)
