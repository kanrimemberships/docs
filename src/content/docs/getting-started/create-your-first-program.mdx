A Program is the activity you offer — "Adult Karate", "Junior Ballet", "Competitive Gymnastics". This is the foundation everything else is built on.

## What this page will cover

- What a program is and how it relates to members and schedules
- Creating a program with a name and description
- Adding classes to a program (the types of sessions within it)
- Adding levels if your school uses a ranking system (belts, grades, etc.)
- Setting level requirements (minimum classes attended, minimum days since last promotion)

## Steps outline

1. Go to Programs → New Program
2. Enter program name (e.g. "Adult Karate")
3. Save and open the program
4. Add classes (e.g. "Beginner Class", "Intermediate Class")
5. Add class variants to each class if needed (e.g. different time slots that count as the same class type)
6. Add levels if applicable (e.g. "White Belt", "Yellow Belt", "Orange Belt")
7. For each level, set eligibility requirements: minimum classes attended, minimum days since last promotion
