Follow this path to go from a brand-new Kanri account to a fully running school.

## What this page will cover

- Prerequisites (what you need before you start)
- The recommended setup order and why it matters
- Links to each step in the process
- Estimated time for initial setup (~30–60 minutes)

## Setup steps (outline)

1. [Create your school](/getting-started/create-your-school) — onboarding wizard + Stripe connection
2. [Configure initial settings](/getting-started/initial-settings) — timezone, tax, membership statuses
3. [Create your first program](/getting-started/create-your-first-program) — set up what you teach
4. [Create your first schedule](/getting-started/create-your-first-schedule) — set up when classes run
5. [Add your first member](/getting-started/add-your-first-member) — enrol a student
6. [Set up billing](/getting-started/set-up-billing) — create a plan and charge your first member
7. [Invite your team](/getting-started/invite-your-team) — bring in staff

## Before you start

- You'll need a Stripe account (or the ability to create one) — Kanri uses Stripe to process payments
- Your school's name, timezone, and address
- Your bank account details for Stripe payouts
