title: Create Your First Schedule
description: How to create a recurring class schedule linked to your program.
---

A Schedule is a recurring class session — "Tuesday Beginner Karate, 6:00–7:00pm". Once created, it generates sessions every week that members can book into and staff can take attendance for.

## What this page will cover

- What a schedule is vs a session vs a booking
- Creating a schedule: name, program, class, day, start/end time, location, capacity
- Activating a schedule so sessions start being generated
- Previewing upcoming sessions after creation
- What to do if you need to pause a schedule (e.g. school holidays)

## Steps outline

1. Go to Schedules → New Schedule
2. Select the program and class this schedule belongs to
3. Set the day of the week, start time, end time
4. Optionally set a location and capacity (max bookings)
5. Save and activate the schedule
6. Verify that upcoming sessions are visible
