## What this page will cover

A reference table for every field on a schedule:

- **Name** — the display name for the schedule (e.g. "Monday Evening Karate")
- **Class** — which class this schedule runs (e.g. "Karate (Blue–Black Belt)"), and optionally which variant (e.g. "Sparring")
- **Recurrence** — how often the schedule repeats (weekly on specific days, etc.)
- **Time** — start time and duration
- **Capacity** — maximum number of bookings allowed
- **Booking window** — overrides the school-wide booking window setting for this schedule
- **Status** — active or inactive (only active schedules generate sessions)
- **Start date / End date** — when the schedule runs from/until

For each field: what it does, what happens if you change it, and when to sync after changes.
