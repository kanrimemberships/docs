## What this page will cover

A reference table for every field when creating or editing a billing plan and its prices:

**Plan fields:**
- Name
- Status (active/archived)

**Price fields:**
- Name/label
- Amount
- Currency
- Billing interval (daily/weekly/fortnightly/monthly/yearly)
- Fixed term (optional — ends the subscription automatically after N periods)
- Status (active/archived)

**Program access fields:**
- Programs (which programs this plan grants kiosk access to)

For each field: what it's for, accepted values, and any gotchas.
