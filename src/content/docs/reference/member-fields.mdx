## What this page will cover

A reference table for every field on the member creation and edit form:

**Basic info:** first name, last name, preferred name, date of birth, gender, photo

**Contact:** email, phone number, address (street, city, state/province, postcode, country)

**School info:** member number, referral source, join date

**Membership (separate tab):** program, status, level, start date, end date, waiver signed

**Notes (separate tab):** free-text notes (mutable)

**Contacts (separate tab):** linked contacts with relationship type and billing contact flag

For each field: what it's for, whether it's required, and any special behaviour.
