## What this page will cover

- What a billing plan is (a named product, e.g. "Monthly Karate", "Annual All-Access")
- The difference between a plan and a price (a plan is the product; a price is the specific amount and interval)
- Single vs multiple price mode (controlled in Settings → Behaviors)
- How prices work: amount, currency, billing interval (daily/weekly/monthly/yearly), optional fixed term
- How the same plan can have prices for different member types (e.g. Adult $120/mo, Junior $80/mo)
- Program access: linking a billing plan to programs so subscription holders get access to sign in for those programs
- Billing plan lifecycle: active plans can be assigned to subscriptions; archiving a plan removes it from future use but doesn't cancel existing subscriptions
