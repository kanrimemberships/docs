## What this page will cover

- What can be edited on a price (name/label)
- What cannot be changed (amount, currency, interval — these are fixed in Stripe once created; create a new price instead)
- How to replace a price: create the new price, migrate existing subscriptions to it, archive the old price
- Archiving a price (hides it from future use, doesn't affect active subscriptions using it)
