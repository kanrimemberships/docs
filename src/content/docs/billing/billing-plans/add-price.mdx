## What this page will cover

- Going to the billing plan and clicking "Add Price"
- Required fields: amount, currency, billing interval
- Optional fields: fixed term (e.g. a 6-month plan that auto-cancels), price name (label for this option, e.g. "Adult Rate", "Junior Rate")
- Billing intervals: daily, weekly, fortnightly, monthly, yearly
- Fixed-term vs ongoing subscriptions
- In single price mode vs multiple price mode — what changes
- Setting a price as the default for new subscriptions
