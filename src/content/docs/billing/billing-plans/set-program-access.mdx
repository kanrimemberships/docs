---
title: Set Program Access on a Billing Plan
description: How to configure which programs a billing plan grants access to (for kiosk sign-in gating).
---

## What this page will cover

- What program access gating is (when enabled in Behaviors, members can only sign in for programs their active subscriptions give them access to)
- How to enable program access gating (Settings → Behaviors → "Use billing plans to gate program access")
- Adding program access to a billing plan: which programs does this plan grant?
- Removing program access from a plan
- What happens if a member has a membership but no subscription with matching program access (they'll be blocked at kiosk)
- When not to use program access (if you don't need access control, keep this off)
