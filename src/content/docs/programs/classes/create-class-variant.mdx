---

## What this page will cover

- What variants are for (distinguishing different styles or formats within the same class — e.g. "Sparring" and "Kata" are both variants of "Karate (Blue–Black Belt)")
- Variants are optional — a class with no variants just runs as itself
- Going to Programs → [Program] → Classes → [Class] → Variants → New Variant
- Naming a variant (e.g. "Sparring", "Kata", "Weapons")
- The "counts towards attendance" flag on a class: if turned off, attending any session of this class (regardless of variant) does not count toward the member's level promotion requirements (e.g. a fun social class that doesn't count toward belt progression)
- When to use variants vs creating a separate class
