## What this page will cover

- What a class is (a named type of training within a program — e.g. "Karate (Blue–Black Belt)", "Beginner Karate", "Open Mat")
- What a class variant is (an optional sub-type of a class — e.g. "Sparring" or "Kata" as variants of "Karate (Blue–Black Belt)"). Variants are optional; a class without variants just runs as itself.
- What a session is (a single scheduled occurrence of a class, optionally with a specific variant, at a specific time — e.g. Monday 7–8pm)
- The relationship: Program → Class → (optional) Variant; Schedule → Sessions
- The "counts towards attendance" flag on the **class** (if off, attending any session of that class, regardless of variant, does not count toward level promotion requirements)
- When to create multiple classes vs multiple variants of one class (different audience/belt range = new class; same class, different style = variant)
