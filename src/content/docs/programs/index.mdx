## What this page will cover

- What a program is (a named activity, e.g. "Adult Karate", "Junior Ballet", "Open Gym")
- How programs are the container for classes, levels, and member memberships
- The relationship: Program → Classes → Schedules → Sessions
- The relationship: Program → Levels → Member Memberships
- Program access gating: how billing plans can control which programs a member can attend (if enabled)
- Viewing all programs and the members enrolled in each
