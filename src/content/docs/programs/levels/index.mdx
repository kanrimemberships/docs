## What this page will cover

- What levels are (optional ranking system within a program — belt grades, skill levels, age groups, etc.)
- How levels are ordered (they are ordered by position within the program)
- Level eligibility requirements: minimum classes attended since last promotion, minimum days since last promotion
- What "counts towards attendance" means for classes (the flag lives on the class — if turned off, attending any session of that class doesn't count toward promotion requirements)
- How promotion works: when a member is promoted their promotion date resets and attendance counter resets
- The Promotions report: how to see who is ready to be promoted
- Schools that don't use levels (levels are optional — programs can have zero levels)
