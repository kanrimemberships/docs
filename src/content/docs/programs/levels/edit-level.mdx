## What this page will cover

- Editing a level name (updates on all existing memberships at that level)
- Changing promotion requirements (only affects future promotion eligibility checks — doesn't retroactively change anyone's status)
- Reordering levels within a program
- Deleting a level (only if no members are currently at that level)
