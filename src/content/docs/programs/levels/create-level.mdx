## What this page will cover

- Going to Programs → [Program] → Levels → New Level
- Required fields: level name (e.g. "White Belt", "Grade 1")
- Setting promotion requirements:
  - Minimum classes attended since last promotion (0 = no minimum)
  - Minimum days since last promotion (0 = no minimum)
- Ordering levels (drag to reorder, or set position)
- What happens if you don't set any requirements (member can be promoted at any time)
