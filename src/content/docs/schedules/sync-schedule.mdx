---
title: Sync a Schedule
description: How to apply changes to a schedule's template to all upcoming sessions.
---

## What this page will cover

- What "Sync" does (takes the current state of the schedule template and updates/activates/deactivates future sessions to match)
- When to use Sync (after editing a schedule's time, class, or location — to push those changes to upcoming sessions)
- What Sync does NOT do (doesn't affect sessions that have already had attendance taken)
- How to trigger a sync from the schedule detail page
- Sync only affects sessions belonging to this schedule (not other schedules)
